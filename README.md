## Project structure

This project serves as a central place to:

- Learn, explore and document C++ topics that I find worth studying in greater depth.
- Explore both fundamental and advanced C++ language features.
- Store code examples that illustrate theoretical concepts.
- Practice problem-solving through exercises.
- Create a long-term reference for future consultation.

To accomplish the purpose of this project, it is structured as follows:

- `topics`: organized notes and examples for specific C++ topics.
- `exercises`: practice exercises and coding challenges.
- `resources`: collection of learning references.

## Topics

The [`topics`](topics/) directory contains the main learning content. Each topic is self-contained and typically is organized in the following way:

```text

├── src/
└── assets/
```

The complete list of available topics is [here](topics/README.md).
